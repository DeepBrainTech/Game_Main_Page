## 注意事项

1. **图片格式**：建议使用 JPG 格式（照片）或 PNG 格式（带透明背景的图片）
2. **文件大小**：请确保图片经过优化，单个文件不超过 500KB，以提高页面加载速度
3. **版权**：确保所有图片都有合法使用权
4. **备用方案**：如果图片不存在，页面会自动显示纯色背景，不会影响功能

## 临时解决方案

在准备好实际图片之前，您可以：
1. 使用占位图片服务（如 placeholder.com）
2. 使用免费图库（如 Unsplash, Pexels）
3. 页面已经设置了优雅的降级方案，即使没有图片也能正常显示

## 更新图片

只需将新图片放入此文件夹，确保文件名与上述列表匹配即可。页面会自动加载新图片。
